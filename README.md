# go-redis-web
redis web admin based on go lang

# build
1. `go get -u github.com/jteeuwen/go-bindata/...`
2. `go get golang.org/x/tools/cmd/goimports`
3. `/gobin.sh`
4. `go build`
5. build for linux :`env GOOS=linux GOARCH=amd64 go build -o go-redis-web.linux.bin`

# startup
`./go-redis-web`
